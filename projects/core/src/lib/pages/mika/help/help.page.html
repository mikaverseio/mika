MikaForm is a powerful, flexible, API-first admin dashboard system built with Angular. It allows you to create admin panels for any backend API without writing custom forms or tables. Instead, MikaForm relies on EntityConfig and global settings to dynamically generate everything.

To begin using MikaForm:

ğŸ” RBAC & Permissions

MikaForm supports field-, action-, and route-level access control:

  • Use permissions inside EntityConfig for each action.
  • Inject user role/permissions using MikaPreferencesService.
  • Route access is protected using MikaPermissionGuard.
