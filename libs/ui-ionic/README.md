# @mikaverse/ui-ionic

> **Status: Deferred to v0.2**

Ionic UI implementation for the Mika admin engine. This package is a placeholder — the Ionic table and form components are stubs.

For a production-ready UI implementation, use [`@mikaverse/ui-material`](../ui-material/README.md).

## Planned for v0.2

- `MikaTableComponent` — Ionic list + card view for mobile
- `MikaFormComponent` — Ionic form fields
- `MikaRootIonicComponent` — Ionic app shell with tabs/sidemenu
- `getMikaUiIonicRoutes()` — router config
- `provideUiIonic()` — DI provider function
