# @mikaverse/ui-i18n

> **Status: Planned for v0.2**

i18n utilities for the Mika admin engine. This package is a stub.

The core i18n functionality (translation loading, language switching, RTL support, `localizable` field handling) is already implemented in `@mikaverse/core` via `MikaI18nService` and `HybridLoader`.

## Planned for v0.2

- Standalone translation file management utilities
- Language detector helpers
- RTL layout utilities
- Integration with additional i18n backends
